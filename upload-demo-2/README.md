# nodejs-upload-demo-web-uploader

## based on

- multer : https://github.com/expressjs/multer
- web-uploader : https://github.com/fex-team/webuploader/

## how to use 

> - clone the code 
> - cd upload-demo-2
> - npm install 
> - npm start 
> - visit localhost:8081
> - the file will upload into /public/uploads
